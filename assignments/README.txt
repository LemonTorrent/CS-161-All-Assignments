assignment5.cpp

This program is a connect 4 game. Users fill a grid with markers and the first person to get 4 of their markers in a row wins the game. This code was written in c++

Build Status:

Features:
The computer chooses a row randomly from the empty rows when playing against a human.

How to Use:
Compile the program with g++. When running the program, include the number of players, the number of rows, and the number of columns. You can choose either 1 or 2 players and between 4-20 rows/columns. From here, imput the number row you would like to place your marker, and enjoy the game!
